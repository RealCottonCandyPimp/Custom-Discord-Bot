XP and levels

Members earn XP from messages in this server. Level is 1 + floor(total XP ÷ XP per level). Use 0 for daily cap to mean no limit. Daily totals reset on UTC midnight.

Level roles

When someone’s level reaches a threshold you add here, the bot assigns that Discord role (same instant as the level-up, from chat XP). The bot’s top role must be above reward roles in Server Settings → Roles, and the bot needs Manage Roles. Paste each role’s numeric ID (Developer Mode → right‑click role → Copy Role ID). Up to 50 rules; each level number must be unique. Use level 2 or higher (everyone starts at level 1).
